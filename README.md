# ConsBlock

> Build consensus at the speed of thought.

Open-source consensus framework that gives distributed systems persistent state and fault-tolerant agreement. Built for Web3, works everywhere.

## 🌐 Website

Live at: [consblock.xyz](https://consblock.xyz) *(coming soon)*

## 🚀 Quick Start

```bash
npx consblock init my-network
```

## ✨ Features

- ⚡ Sub-second Finality — <50ms BFT block finality
- 🔐 Cryptographic State — Merkle-trie backed, tamper-proof
- 🌐 Peer Discovery — libp2p-powered, no central bootstrapper
- 🔄 Hot-swappable Engines — PBFT, Tendermint, or custom
- 🧩 EVM Compatible — drop-in JSON-RPC replacement
- 📊 Built-in Observability — Prometheus + structured logs
